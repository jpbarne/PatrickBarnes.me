2-D Game Development Final Project

Written in C++ and using the SDL2 library, my final project is based on classic asteroid shooter arcade games. I implemented some additional features to add my own spin on the game by including powerups (triple-shot, rapid fire, and shield), and waves of enemies. Credit to Mitchell Bowser for all sprites used and Dr. Malloy for the framework.
